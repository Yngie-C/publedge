# 오퍼 C — 인터랙티브 전환 대행 영입 레터

## 용도

**대상**: 위키독스에 완결된 책이 있지만 판매량·추천수가 낮은 underdog 저자. 집필에 더 투자할 여력이 없거나, 본업 바쁜 저자.
**제안**: 원고는 그대로 가져오고, 저희가 **인터랙티브 전환·표지·판매 페이지·마케팅**을 전담. 저자는 검수만.
**핵심 메시지**: "당신 시간 최소 투입. 나머지는 우리가 전부."

## 오퍼 C 특유의 설득 구조

**이 오퍼의 감정적 핵심**: "당신은 이미 충분히 일했고, 나머지는 우리가 합니다"

- underdog 저자들은 보통 "내 책이 왜 안 팔릴까?" 고민하고 있지만, 그걸 직접 해결할 에너지·시간이 없음
- "저희가 대신 해드립니다"가 감정적으로 가장 크게 꽂히는 메시지
- 단, **리스펙트가 없으면 "저작권 뜯어가려는 놈"으로 보임** → 저자 원고에 대한 존중이 메시지 전반에 깔려야 함

## 오퍼 C의 가장 큰 실패 지점

**실패 유형 1**: 인터랙티브 전환 초안이 저자 취향과 안 맞아서 검수에서 무한 반려

**예방책**:
1. 킥오프 때 저자와 함께 "이런 체크리스트는 안 어울린다", "이런 워크시트는 좋다" 사전 취향 조율
2. 첫 장만 먼저 전환해서 보여주고 방향 합의 후 나머지 진행
3. 반려 3회 이상 시 "오퍼 A로 전환할지 종료할지" 솔직한 대화

**실패 유형 2**: 저자가 "이게 내 책 맞나?" 이질감 호소

**예방책**:
1. 저자 문체·어조를 그대로 보존 (인터랙티브 레이어만 추가)
2. 표지 디자인도 저자 취향 반영 (3안 제시 후 선택)
3. 저자 이름 브랜딩 존중 (플랫폼 전면화보다 저자 이름 전면화)

**실패 유형 3**: 마케팅 약속 못 지킴 (첫 100권 판매 책임)

**예방책**:
1. 첫 100권 판매는 "플랫폼 마케팅 예산 확보 후 보장" 전제로 계약서 조정
2. 구체 수치 약속 어려우면 "첫 달 론칭 주간 마케팅 확정"으로 조정
3. 초기 3~5명까지는 실패해도 배상 없는 실험 계약 (저자에게 솔직히 고지)

## 오퍼 A/B/C 판별 플로우 (어떤 레터 쓸지)

```
저자 프로파일 확인
├─ 이미 종이책 출판 계약? → 모두 제외
├─ 완결 책 있음 + 추천수 높음 (>300) + 전자책 판매 중
│   → 오퍼 A (프리미엄 개정판)
├─ 완결 책 있음 + 실무 배경 명확 + 블로그/링크드인 활발
│   → 오퍼 B (신규 업무 노하우)
├─ 완결 책 있음 + 판매 저조 + 저자 활동 적음
│   → 오퍼 C (인터랙티브 전환 대행)
└─ 미완결 / 좀비 책 / 번역서
    → 컨택 보류
```

**동시 제안 가능한가?** — 가능하지만 첫 컨택에선 한 가지만. 대화 중에 "이것도 있어요, 저것도 있어요" 가능.
