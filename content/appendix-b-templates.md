# 부록 B: 프롬프트 템플릿 모음

자소서 작성을 도와줄 프롬프트 템플릿을 한곳에 모아뒀어요. Claude Code 채팅창에 그대로 붙여넣거나, `.claude/commands/` 폴더에 스킬 파일로 저장해서 사용하세요. 각 템플릿은 복사-붙여넣기 바로 가능하도록 정리했어요.

---

## 자소서 항목별 강화 프롬프트

자소서의 각 항목을 더 설득력 있게 다듬고 싶을 때 사용하는 템플릿이에요.

### 지원동기 강화

단순히 "관심이 있어서"로 끝나는 지원동기를 회사 맞춤형으로 바꿔줘요.

```
현재 자소서의 지원동기를 강화해줘.
회사의 최근 프로젝트나 뉴스를 반영하고,
나의 구체적 경험과 연결해줘.
단순히 '관심이 있어서'가 아니라
'왜 이 회사여야 하는지'가 명확히 드러나도록 작성해줘.
```

> **TIP:** 이 프롬프트를 쓰기 전에 "이 회사의 최근 뉴스나 주목할 만한 프로젝트를 알려줘"라고 먼저 물어보세요. Claude가 컨텍스트를 갖고 있으면 훨씬 구체적인 지원동기를 만들어줘요.

### 직무역량 강화

JD와 내 프로필을 연결해서 역량 섹션을 강화해요.

```
직무역량 부분을 강화해줘.
내 프로필(data/my-profile.md)에서 JD 필수 역량과 매칭되는 경험을 찾아서,
구체적 수치와 성과를 포함해줘.
STAR 기법(상황-과제-행동-결과) 구조로 작성해줘.
```

### 성장과정 STAR 변환

막연한 서술형 성장과정을 구조적으로 바꿔줘요.

```
성장과정을 STAR 기법으로 다시 써줘.
- 상황(Situation): 어떤 문제나 환경이었는지
- 과제(Task): 내가 맡은 역할과 목표
- 행동(Action): 구체적으로 한 것
- 결과(Result): 정량적 성과와 배운 점
순서를 지키되, 자연스러운 문장으로 연결해줘.
```

### 입사 후 포부 구체화

"열심히 하겠습니다"에서 벗어나 단계별 성장 계획을 담아줘요.

```
입사 후 포부를 더 구체적으로 써줘.
1년차에는 어떤 기술을 익히고,
3년차에는 어떤 프로젝트를 이끌고 싶은지,
JD의 우대 사항과 연결해서 작성해줘.
회사에 기여하는 방향으로 마무리해줘.
```

## JD 분석 변형 프롬프트

JD 분석 결과를 더 다양하게 활용하고 싶을 때 써요.

### 경쟁사 비교 분석

지원동기에 쓸 회사의 차별점을 찾아줘요.

```
이 회사와 같은 업계 경쟁사를 3개 알려주고,
이 회사만의 차별점을 분석해줘.
자소서 지원동기에 활용할 수 있는 구체적인 포인트를 3~5개 뽑아줘.
```

### 면접 예상 질문 생성

자소서를 제출한 후 면접 준비까지 이어서 할 수 있어요.

```
이 JD 분석 결과와 내 자소서 내용을 바탕으로
면접에서 나올 수 있는 질문 10개를 만들어줘.
자소서에서 약점이 될 수 있는 부분에 대한 질문도 포함해줘.
각 질문에 어떻게 답하면 좋을지 힌트도 한 줄씩 붙여줘.
```

> **TIP:** 면접 예상 질문은 자소서 초안이 완성된 후에 사용하세요. Claude가 자소서 내용을 참고해서 "이 부분을 더 파고들 것 같다"는 위험 질문까지 짚어줘요.

### 회사 문화 심층 분석

JD 텍스트에서 조직문화 코드를 읽어내줘요.

```
이 JD에서 회사의 조직문화, 일하는 방식, 핵심 가치관을 추론해줘.
'수평적 문화', '애자일', '데이터 드리븐' 같은 키워드를 찾고,
각 키워드가 자소서 어느 부분에서 반영되면 좋을지 알려줘.
```

## 자소서 품질 향상 프롬프트

초안이 완성된 후 퀄리티를 한 단계 올리는 프롬프트예요.

### 뻔한 표현 제거

인사 담당자가 하루에 수백 번 읽는 문구를 걷어내줘요.

```
이 자소서에서 '열정', '성장', '도전', '최선', '노력' 같이
뻔하거나 추상적인 표현을 모두 찾아줘.
각각을 구체적인 에피소드나 수치로 대체한 버전을 제안해줘.
```

### 글자 수 조정

기업별로 다른 글자 수 제한을 맞출 때 사용해요.

```
이 자소서의 [항목명] 항목을 정확히 [N]자로 맞춰줘.
핵심 내용과 수치는 유지하면서 길이만 조정해줘.
조정 후 실제 글자 수도 알려줘.
```

> **TIP:** `[N]`에 원하는 글자 수를, `[항목명]`에 "지원동기" 또는 "직무역량"처럼 항목 이름을 넣으면 돼요. 전체 자소서를 한 번에 조정하려면 "모든 항목을 각각 500자로" 처럼 요청하세요.

### 톤 변경

회사 분위기에 맞게 문체를 바꿔줘요.

```
이 자소서의 톤을 더 [전문적인 / 열정적인 / 차분하고 논리적인] 방향으로 수정해줘.
내용과 구체적인 수치는 그대로 유지하고,
문체와 단어 선택만 바꿔줘.
```

## 기타 유용한 프롬프트

취업 활동 전반에서 활용할 수 있는 확장 템플릿이에요.

### 포트폴리오 README 생성

GitHub 프로필을 자소서와 일관되게 만들어줘요.

```
data/my-profile.md를 바탕으로 GitHub 포트폴리오 README.md를 만들어줘.
다음 항목을 포함해줘:
- 기술 스택 배지 (shields.io 형식)
- 대표 프로젝트 3개 소개 (역할, 기술, 성과)
- 자소서와 일관된 한 줄 자기소개
- 이메일 또는 LinkedIn 연락처
마크다운 형식으로 완성본을 바로 줘.
```

### LinkedIn 요약 생성

영어권 채용 플랫폼을 위한 About 섹션을 만들어줘요.

```
data/my-profile.md를 바탕으로 LinkedIn 프로필 About 섹션을 영문 300자 이내로 작성해줘.
핵심 역량 2가지, 대표 성과 1가지, 앞으로의 방향성을 담아줘.
너무 딱딱하지 않고 자연스러운 영어 문체로 써줘.
```

### 영문 커버레터 변환

한국어 자소서를 영문 커버레터로 재구성해줘요.

```
이 한국어 자소서를 영문 커버레터(Cover Letter) 형식으로 변환해줘.
한국식 구조(지원동기 / 직무역량 / 성장과정 / 포부)를
영문 커버레터 구조(Opening / Body / Closing)로 재구성해줘.
분량은 A4 한 페이지(350~450 단어) 이내로 맞춰줘.
```

> **TIP:** 영문 커버레터는 한국 자소서보다 훨씬 간결해요. "저는 어려서부터..."로 시작하는 성장과정은 생략하고, 직무역량과 지원동기 중심으로 재편하는 게 맞아요. Claude가 이 구조 변환까지 자동으로 해줘요.

## 프롬프트 잘 쓰는 법

템플릿을 그대로 쓰는 것도 좋지만, 조금만 다듬으면 결과물 품질이 크게 달라져요.

### 좋은 프롬프트의 공식

```
[역할 설정] + [구체적 지시] + [제약 조건] + [출력 형식]
```

예를 들어 볼게요.

나쁜 예:
```
자소서 잘 써줘.
```

좋은 예:
```
신입 개발자 자소서의 직무역량 항목을 써줘.
STAR 기법을 사용하고, 수치를 최소 2개 포함해줘.
글자 수는 500자 이내로 맞춰줘.
```

> **TIP:** "잘 써줘"보다 "STAR 기법으로, 300자 이내로, 수치를 포함해서 써줘"가 훨씬 구체적이에요. Claude는 제약 조건이 명확할수록 더 좋은 결과를 내줘요.

### 프롬프트를 스킬 파일로 저장하기

자주 쓰는 프롬프트는 매번 타이핑하는 대신 스킬 파일로 저장해두세요.

```bash
# .claude/commands/ 폴더에 저장하면 /명령어로 바로 호출할 수 있어요
.claude/commands/strengthen-motivation.md   # 지원동기 강화
.claude/commands/adjust-length.md           # 글자 수 조정
.claude/commands/remove-cliches.md          # 뻔한 표현 제거
```

> **TIP:** 스킬 파일 이름은 영문 소문자와 하이픈(-)으로 작성하는 게 관례예요. `/strengthen-motivation` 처럼 슬래시를 붙여서 Claude Code에서 바로 호출할 수 있어요.

### 반복 사용 패턴 만들기

자소서 파이프라인을 한 번 구축해두면 다음 지원할 때 이렇게 사용할 수 있어요.

```
1. JD 파일 업데이트 → /analyze-jd 실행
2. 강화 프롬프트 순서대로 적용
3. /check-quality 로 품질 점검
4. 글자 수 조정 후 최종 완성
```

파이프라인이 익숙해지면 한 회사당 자소서 작성 시간을 30분 이내로 줄일 수 있어요. 처음에는 시간이 걸리더라도 한 번 제대로 세팅해두면 그 이후가 훨씬 편해져요.
