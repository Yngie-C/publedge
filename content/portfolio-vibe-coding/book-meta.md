# 대화만으로 만드는 나만의 포트폴리오 사이트

## 책 정보

- **제목**: 대화만으로 만드는 나만의 포트폴리오 사이트
- **저자**: inspic
- **출판 플랫폼**: Publedge
- **출판일**: 2026년
- **카테고리**: IT/기술, 취업/자기계발
- **대상 독자**: 취준생, Vibe Coding 입문자
- **예상 독서 시간**: 약 1.5~2시간

## 책 소개

Claude Code와 **대화만으로** 나만의 HTML/CSS 포트폴리오 사이트를 만들고, Vercel로 공개 배포하는 실전 가이드예요.

터미널을 한 번도 써본 적 없는 완전 초보자도 이 책을 따라하면:
- 이름·사진·한줄소개가 담긴 명함 페이지를 만들 수 있어요
- 프로젝트 카드 3장 그리드를 직접 올릴 수 있어요
- 색·폰트·여백을 내 취향으로 골라 바꿀 수 있어요
- 모바일 반응형 + 다크모드를 손에 넣을 수 있어요
- 이력서에 붙일 **공개 URL 1개**와 **GitHub 레포**를 가져갈 수 있어요

기존 AI 포트폴리오 빌더(Framer, Wix AI 등)와 달리, 이 책에서는 **내가 쓴 말이 그대로 반영되는 대화형 개발**을 경험해요. 복잡한 설정이나 여러 파일 사이를 오가는 일 없이, 대화창에 원하는 것을 말하면 Claude가 파일을 직접 고쳐줘요. 즉시 보이는 결과물, 낮은 실수 비용 — 이게 첫 Vibe Coding 경험을 가볍게 만들어주는 이 책의 핵심이에요.

## 이 책을 어떻게 읽을까

### 자소서 책 독자에게

『클로드 코드로 만드는 나만의 자소서 파이프라인』을 이미 읽으셨다면, 이 책은 **한 번 더 가볍게 대화형 개발을 체험하는 자매편**이에요. 스킬 저작이나 파이프라인 설계 없이, "대화만"으로 눈에 보이는 결과물을 쌓아가는 감각을 다시 한 번 즐길 수 있어요. Ch0 설치 파트는 익숙하시면 빠르게 넘기셔도 괜찮아요.

### 처음 Vibe Coding 시도하는 독자에게

터미널이 처음이어도 괜찮아요. Ch0이 설치부터 첫 `claude` 실행까지 함께 걸어드려요. 자소서 책을 읽지 않아도 이 책 한 권으로 충분히 시작할 수 있어요.

## 목차

0. 준비: 터미널 공포 풀기 + 도구 설치
1. 첫 결과물: "안녕 세상, 제 이름은 ___입니다"
2. 프로젝트 섹션 추가: 카드 3장 그리드
3. 감성 업그레이드: 색·폰트·여백
4. 환경에 적응하기: 모바일 반응형 + 다크모드
5. 세상에 공개: GitHub + Vercel
- 부록 A: 트러블슈팅 가이드
- 부록 B: 프롬프트 템플릿 모음
- 부록 C: 더 알아보기

## 사전 준비물

1. **Claude Pro 또는 Max 구독** (월 $20~$100) — claude.ai에서 가입
2. **GitHub 계정** (무료) — github.com에서 가입
3. **Vercel 계정** (무료) — vercel.com에서 가입
4. **본인 텍스트** — 이름, 한줄소개, 이력서에 올릴 프로젝트 3개

## 기술 기준

- Claude Code v1.x (2026년 기준)
- Node.js 18 이상
- macOS / Windows 모두 지원
- 순수 HTML + CSS + 최소 JS (빌드 스텝 없음)
