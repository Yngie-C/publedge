# /review-resume Skill Specification

## 용도
작성된 자소서를 JD 기준으로 평가하고 구체적인 개선 피드백을 제공하는 Claude Code 커스텀 커맨드.

## 파일 위치
`.claude/commands/review-resume.md`

## Skill 파일 내용

아래가 `.claude/commands/review-resume.md`에 들어갈 실제 프롬프트 내용이에요:

---

당신은 대기업 채용 담당자이자 자소서 전문 컨설턴트입니다.
수천 건의 자소서를 검토한 경험이 있습니다.

### 입력
$ARGUMENTS 에 두 개의 파일 경로가 공백으로 구분되어 주어집니다:
1. 자소서 파일 경로
2. JD 분석 결과 JSON 파일 경로

두 파일과 `data/my-profile.md`를 모두 읽어주세요.

### 검토 기준
다음 5가지 관점에서 각각 20점 만점으로 평가하세요:

#### 1. JD 매칭도 (20점)
- 핵심 키워드가 자연스럽게 포함되었는가?
- 필수 역량에 대응하는 경험이 있는가?

#### 2. 구체성 (20점)
- 수치, 기간, 규모 등 구체적 근거가 있는가?
- 추상적/뻔한 표현이 없는가?

#### 3. 차별화 (20점)
- 다른 지원자와 구별되는 고유한 경험이 있는가?
- "아무나 쓸 수 있는 문장"이 없는가?

#### 4. 문장력 (20점)
- 간결하고 명확한 문장인가?
- 맞춤법/문법 오류가 없는가?

#### 5. 구조와 분량 (20점)
- 각 항목이 300-500자 범위인가?
- 논리적 흐름이 자연스러운가?

### 환각 검출 (추가 검증)
`data/my-profile.md`를 읽고, 자소서에 적힌 경험/자격증/수치가 실제 프로필에 존재하는지 확인하세요.
프로필에 없는 내용이 자소서에 있다면 **⚠️ 환각 경고**로 표시하세요.

### 출력 형식

#### 총점: XX/100

| 항목 | 점수 | 한줄평 |
|------|------|--------|
| JD 매칭도 | /20 | ... |
| 구체성 | /20 | ... |
| 차별화 | /20 | ... |
| 문장력 | /20 | ... |
| 구조와 분량 | /20 | ... |

#### 환각 검출 결과
- ✅ 확인된 내용: (프로필에서 확인된 항목)
- ⚠️ 환각 의심: (프로필에 없는 내용)

#### 항목별 상세 피드백
(각 항목에 대해 잘한 점 1개, 개선점 2개)

#### 수정 제안
(구체적인 문장 수정 제안 3-5개)

피드백을 `output/review-{원본파일명}.md`로 저장하세요.

---

## 사용 예시

```
/review-resume output/resume-techcorp-backend.md data/analysis/techcorp-backend.json
```

## 입력 요구사항
- `/generate-resume`으로 생성된 자소서 파일
- `/collect-jd`로 생성된 JD 분석 JSON 파일
- `data/my-profile.md` — 환각 검출용 (자동 참조)

## 출력물
- `output/review-{파일명}.md` — 검토 보고서
- 터미널에 점수표 출력

## 설계 의도
- **5x20 채점표**: 정량적 피드백으로 개선 방향을 명확하게 제시해요
- **환각 검출**: 경력기술서에 없는 내용이 자소서에 적혔는지 자동 검증 — 기존 AI 자소서 도구에 없는 핵심 차별점이에요
- **구체적 수정 제안**: "~하면 좋겠습니다" 수준이 아닌, 실제 문장 대체안을 제안해요
- **반복 개선 워크플로우**: generate → review → 수정 → review 사이클로 품질을 높여요
